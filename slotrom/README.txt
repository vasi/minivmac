SlotRom: READMEPaul C. PrattDecember 1, 2011SlotRom is a utility to write out to an image file thedeclaration ROM of each Nubus Card in some Macintoshmodels.Further information may be found at"http://www.gryphel.com/c/minivmac/extras/slotrom/index.html".You can redistribute SlotRom and/or modify it under the termsof version 2 of the GNU General Public License as published bythe Free Software Foundation.
